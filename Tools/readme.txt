Tools that are compatible with NinjaParty.

G2DF - 3pn fork of GLEED2D
	GLEED2D and G2DF both use the Ms-PL (Microsoft Public License).
	Windows only (works in Parallels on Mac OS X)
	Requries .Net 3.5 and XNA 3.1
	Gleed source - http://gleed2d.codeplex.com/
	G2DF source - http://svn2.xp-dev.com/svn/G2DF/
	Notes on G2DF - http://thirdpartyninjas.com/blog/2010/11/23/g2df/

Demina - Skeletal animation tool
	Microsoft Public License
	Windows only (works in Parallels on Mac OS X)
	Requires .Net 4.5 and XNA 4.0
	Source - http://demina.codeplex.com/

Spine - Skeletal animation tool
	Cross platform
	$55 to license (as of Apr 2, 2013)
	http://esotericsoftware.com/

Glyph Designer - Font tool
	Mac OS X only
	$39.99 to license (as of Apr 2, 2013)
	http://www.71squared.com/en/glyphdesigner

Texture Packer
	Cross platform
	Works with free version, but Pro version recommended
	$29.95 to license Pro version (as of Apr 2, 2013)
	http://www.codeandweb.com/texturepacker
	Note: Choose Json-Array export format.
